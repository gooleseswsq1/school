## ⌨️ Keyboard Shortcuts (Master These!)

### Essential Shortcuts

| Key | Action | When to Use |
|-----|--------|------------|
| `Delete` | Remove selected element | Delete after selecting |
| `Ctrl+Z` (Win) / `Cmd+Z` (Mac) | Undo last change | Oops! Redo mistake |
| `Ctrl+C` (Win) / `Cmd+C` (Mac) | Duplicate element | Clone the selected item |
| `↑` `↓` `←` `→` | Move 1 pixel | Fine-tune position |
| `Shift + Arrow` | Move 10 pixels | Faster movement |
| Mouse Wheel | Zoom in/out | Change size view |

### Examples

**Workflow 1: Quick Delete**
```
1. Click element to select
2. Press Delete key
3. Element gone!
```

**Workflow 2: Copy & Paste**
```
1. Click element
2. Press Ctrl+C
3. Element duplicated nearby
4. Use arrows to move it
```

**Workflow 3: Undo Mistakes**
```
Made a mistake?
Press Ctrl+Z (up to 10 times)
```

## 🎨 How to Use Common Features

### Adding Text

```
Step 1: Click [T] button (left panel) or press the key
Step 2: A text box appears on canvas with "Thêm văn bản..."
Step 3: Click to select the text and type your own
Step 4: Click outside the text box to finish editing
Step 5: Click the text again to format it
```

**Result**: Text box with formatting in bottom-right panel

### Adding an Image

```
Step 1: Click [🖼] button (left panel)
Step 2: Choose an image from your computer
Step 3: Image appears on canvas
Step 4: Drag corners to resize
Step 5: Click and drag to move to any position
```

**Tips**:
- Images start at 250px wide
- Drag anywhere on it to move
- Select and press Delete to remove

### Editing Text Properties

```
Step 1: Click on any text to select it
Step 2: Bottom-right panel appears with 3 sections:

   COLOR: Click square to pick text color
   SIZE: Drag slider (8px - 72px) or type
   ALIGN: Click [←] [·] [→] for left/center/right
```

**Keyboard Tip**: Don't click color picker, just type RGB values

### Changing Canvas Color

```
Method 1: Click color square (bottom-left)
          Select new color → Canvas updates instantly

Method 2: Left panel > Màu nền > Pick from 8 colors
          Click any color square → Canvas changes
```

## 🔥 Pro Tips & Tricks

### Tip 1: Fine Positioning
```
Don't drag elements slowly - use arrow keys!
Select element → Hold arrow keys → Precise movement
Shift + arrow = 10px jumps (faster)
```

### Tip 2: Quick Zooming
```
Don't use zoom buttons - scroll your mouse wheel!
Scroll up = Zoom in
Scroll down = Zoom out
Fast and intuitive!
```

### Tip 3: Fit Whole Design
```
If canvas is too big/small:
Click [□] button (right panel)
Canvas auto-scales to fit screen perfectly
```

### Tip 4: Undo Accidentally Deleted
```
Oops! Deleted something important?
Press Ctrl+Z (then Ctrl+Z again if needed)
History goes back 10 actions max
```

### Tip 5: Duplicate Complex Designs
```
Made an element with perfect styling?
Select it → Press Ctrl+C → Instant copy created
Appears slightly offset, ready to move
No click-drag needed!
```

## ⚠️ Common Mistakes to Avoid

### ❌ Mistake 1: Forgetting to Click Outside Text
```
You type text... but it never appears on canvas?
→ Click OUTSIDE the text box to finish editing
→ Only then can you select it for formatting
```

### ❌ Mistake 2: Trying to Move Before Selecting
```
You can't drag text that's already being edited
→ Click outside first to exit edit mode
→ Then click again to select (not edit)
→ Now you can use arrows to move
```

### ❌ Mistake 3: Canvas Too Small to Work
```
"I can't see my design!"
→ Use Scroll wheel to zoom in
→ Or click [□] button to fit screen
→ Don't maximize browser window, use zoom instead
```

### ❌ Mistake 4: Color Picker Seems Hidden
```
"Where's the color button?"
→ Look bottom-left with paint palette icon 🎨
→ Click the colored square next to it
→ Should open color picker
```

## 🆘 Troubleshooting

### "Button is disabled/grayed out"
**Reason**: No object selected on canvas
**Fix**: Click on the element you want to edit

### "Text won't format"
**Reason**: Still in edit mode (blinking cursor visible)
**Fix**: Click outside text box first, then select again

### "Canvas looks tiny"
**Reason**: Zoom level is low
**Fix**: Click [□] Fit Screen or scroll wheel to zoom

### "Can't see floating panels"
**Reason**: They're outside visible area
**Fix**: Click on canvas and scroll, or maximize window

### "Design disappeared"
**Reason**: Accidentally deleted it
**Fix**: Press Ctrl+Z multiple times (up to 10 undos)

### "Downloaded image is low quality"
**Reason**: Zoom was very low when downloading
**Fix**: Set zoom to 100%+ before downloading
