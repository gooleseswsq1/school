## 🎯 5-Minute Quick Start

### Step 1: Start the server
```bash
npm run dev
```

### Step 2: Create test data
```bash
node prisma/seed-quiz.js
```

### Step 3: Login
- Go to http://localhost:3000
- Login as: `teacher@example.com`

### Step 4: Create a quiz
- Click "Editor" or "Tạo mã kích hoạt"
- Select "Bài Giảng C Programming" page
- Click "Quiz ✨" button at bottom
- Click "Thêm câu hỏi"
- Type a question
- Click "Lưu Quiz"

### Step 5: Try as student
- Logout and login as: `student@example.com`
- Open the lesson to see your quiz
- Select answers
- Click "Kiểm tra đáp án"

✅ Done in 5 minutes!
